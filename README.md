## ✨ 特性亮点

|     |     |
| --- | --- |
| 🚀 **25+ 实用工具** | 覆盖 JSON、图片、音视频、文档、编码、开发六大分类 |
| 🖥️ **Web + 桌面双端** | 一套代码，同时支持浏览器访问和 Electron 桌面应用 |
| 🌙 **深色商务主题** | 内置暗色模式，护眼且专业，长时间使用不疲劳 |
| ⚡ **纯前端实现** | 所有工具均在浏览器/本地运行，数据绝不离开你的电脑 |

| 🌍 **中英文国际化** | 内置 i18n 支持，无缝切换中文 / English |

## 📦 安装与使用

```bash
# 克隆仓库
git clone https://github.com/your-org/tool-hub.git
cd tool-hub

# Web 开发模式
npm install && npm start

# 桌面开发模式（Electron）
npm run electron:dev

# 打包桌面应用
npm run build && npx electron-builder
```

## 📂 项目结构

```
tool-hub/
├── src/
│   ├── tools/              # 25 个工具组件，按分类组织
│   │   ├── json/           # JSON & 数据相关
│   │   ├── image/          # 图片处理相关
│   │   ├── media/          # 音视频相关
│   │   ├── document/       # 文档 & 办公相关
│   │   ├── encode/         # 编码 & 加密相关
│   │   └── devtools/       # 开发者工具
│   ├── components/         # 公共组件（Sidebar、HomePage、ToolCard）
│   ├── i18n/               # 中英文国际化配置与词条
│   ├── electron/           # Electron 主进程与预加载脚本
│   ├── styles/             # 全局样式与主题变量
│   ├── hooks/              # 自定义 Hooks
│   └── utils/              # 工具函数库
├── public/                 # 静态资源
├── screenshots/            # 界面截图
├── demos/                  # 功能演示 GIF
├── electron-builder.yml    # Electron 打包配置
├── package.json
├── tsconfig.json
├── vite.config.ts
└── README.md
```
